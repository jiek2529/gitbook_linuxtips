# yum

## 1. 安装.installing
1. 去下载http://ayo.freshrpms.net/相应的rpm包
+ 安装yum > sudo rpm -ivh {*.rpm}
+ > yum update # 更新
+ > yum install {pkg} #安装包
+ > yum clean # 清除无用程序包
+ > yum remove # 卸载RPM程序

## 2. 使用used
1. > yum install <pkg_name> #安装程序
+ > yum remove <pkg_name>
+ > yum search <keyword>
+ > yum list
+ > yum list updates
+ > yum list installed
+ > yum list extras
+ > yum list <pkg_name>
